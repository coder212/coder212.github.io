Cara Menggunakan Perintah Mount

Published: Rab 23 Maret 2016
By arba

In Tutorial.

Apa Itu Mount? Mount adalah memuat device kedalam filesistem sistem operasi, mount kita perlukan untuk memuat device - device penyimpanan kedalam filesistem gnu/linux sistem operasi agar kita bisa mengakses isi dari device-device tersebut.

lalu bagaimanakah cara memuat atau mount device tersebut begini caranya

code:mount -t type device tujuan

contoh:

code:mount -t ntfs /dev/sdb2 /media/username
code:mount /dev/sda1 /

itu artinya memuat /dev/sda1 sebagai root(/) filesistem biasanya kalo langsung seperti itu menggunakan type ext.

code:mount -t proc none /mnt/proc

perintah ini digunakan untuk memuat isi dari /proc system host kedalam system tamu.

code:mount -o bind /sys /mnt/sys

perintah ini digunakan untuk mereplikasi isi dari /sys host kedalam /sys tamu.
