Cerita Tentang Kehidupan GNU/Linux Enthusiast

Saya pertama kali mengenal GNU/Linux tahun 2010 pada waktu kelas dasar pemrograman. Sejak saat itu saya jadi penasaran apa itu GNU/Linux, mengapa GNU/Linux dan lain sebagainya. pada ahirnya saya mencoba Ubuntu 10.10 ketika menggunakan Ubuntu 10.10 saya mengalami kebingungan karena file audio saya yang kebetulan berformat mp3 tidak dapat diputar di Ubuntu saya, bahkan saya tidak dapat menggunakan printer saya padahal saya sedang mengerjakan tugas yang membutuhkan printer untuk mencetak tugas yang sedang saya kerjakan. Hal ini memaksa saya untuk mempelajari teknis bagaimana printer bekerja apa saja yang harus saya penuhi agar printer dapat bekerja saya mencari drivernya untuk ubuntu dan lain sebagainya setelah berjam-jam mencari cara untuk dapat membuat printer saya berfungsi dengan baik, ahirnya saya menemukannya dan printer saya sudah berfungsi normal tapi masalah file audio yang berformat mp3 masihlah belum ditemukan solusinya.

Setelah berjam-jam mencari solusi sampai bingung sendiri belum ketemu juga solusinya sampai saya putus asa, setelah putus asa itulah saya iseng memutar berkas suara yang berformat mp3 dalam posisi terhubung ke internet ternyata sang media playernya mencari sendiri kodeknya selesai sudah semua persoalan tersebut, meskipun ketika berkenalan dan menggunakan GNU/Linux mengalami banyak masalah tapi saya tidaklah punya keinginan sedikitpun kembali menggunakan windows ataupun macos. Tak berapa lama saya tertarik untuk menginstall sabily alquds yang menurutku sangat enak karena applikasi yang bisa dipergunakan untuk beribadah sudah terinstall didalamnya menggunakan sabily saya hampir tak mengalami hambatan satupun dengan sabily inilah saya mengeksplorasi tampilannya mulai dari compiz merubah tampilan ala xp, win7 bahkan macos sempat saya lakukan dan itu semua mengajarkan saya untuk terus berkembang.

Dengan sabily pulalah saya membuat applikasi untuk android, karena saya penasaran Ubuntu 11.04 ahirnya saya upgrade sabily saya menjadi 11.04, dengan sistem ini saya mengeksplorasi python dan nasm assembler. Ketika muncul Ubuntu 11.10 saya update dan upgrade lagi, di system ini saya banyak mengeksplorasi bash script, semua script saya buat untuk memenuhi keingintahuan saya mengenai bash bisa dibuat untuk apa, setelah ada laptop baru untuk menunjang pekerjaan laptop tersebut menggunakan Ubuntu 12.04, netbook saya install backtrack5 demikian ahir dari netbook favorit saya hehe setelah itu saya tak pernah menyentuh netbook saya. dengan laptop inilah saya berpetualang kebanyak distro. distro tersebut adalah sebagai berikut:

  1. Ubuntu 12.04.
  2. BodhiLinux.
  3. Kali Linux.
  4. ArchLinux.
  5. Freebsd ( Unix like sebenarnya bukan distro GNU/Linux ).
  6. Slackware.
  7. CRUX.
  8. Kwort.
  9. LFS.
  10. OpenSUSE.
  11. Gentoo.

Ahirnya saya memutuskan untuk bertahan di Gentoo dengan ditemani oleh archlinux karena hanya dua sistem itu yang pas dihati, meskipun suatu hari nanti saya akan kembali menggunakan LFS tapi setidaknya saya saat ini cukup digentoo didalam gentoo ini saya mencoba berpetualang dengan golang sebuah bahasa pemrograman yang dibuat oleh programmer google, bahasa ini sangat enak membuat source code dengan menggunakan golang, dan sayapun masih akan tetap mengeksplorasi bahasa pemrograman c dan assembly yang 64bit. Cukup ini yang bisa saya ceritakan tentang kehidupan saya dalam menggunakan GNU/Linux ada suka dan duka tapi itulah GNU/Linux sistem yang bebas untuk dieskplorasi.
