Cara Install Gentoo

Published: Rab 16 Maret 2016
By arba

In Tutorial.

tags: gnu/linux install gentoo

Cara install gentoo versi saya hehe pastikan kamu memiliki distro gnu/linux lainnya atau kamu menggunakan liveusb atau livecd boleh boleh saja dalam hal ini saya menggunakan distro ubuntu 14.04 sebagai hostnya pertama lakukan hal ini

code:sudo apt-get install dchroot debootstrap

akan digunakan untuk melakukan chroot ke system gentoo nantinya. Berikut ini langkah-langkahnya //download stage3 dan portage

  1. buat partisi home dan / dengan nama /dev/sdb2 /dev/sdb3 ---> dalam hal ini flashdisk saya kalo hardisk saya /dev/sda sesuaikan dengan /dev/sd* milikmu bisa sdc,sdd,sde,sdf dll jadi usahakan ls /dev/sd* , kemudian sudo -i sebagai root.
  2. mount partisi tersebut.
code:mount /dev/sdb2 /mnt/gentoo
code:mount /dev/sdb2 /mnt/gentoo/home
  1. masuk kedalam direktori /mnt/gentoo.
code:cd /mnt/gentoo

4.download stage3 dan portage dengan menggunak wget.

untuk unduh stage3

code:wget -t 0 http://distfiles.gentoo.org/releases/amd64/autobuilds/current-stage3-amd64/stage3-amd64-20160310.tar.bz2

untuk 64bit dan kalo 32 bit

code:wget -t 0 http://distfiles.gentoo.org/releases/x86/autobuilds/current-stage3-i686/stage3-i686-20160308.tar.bz2

unduh portage

code:wget -t 0 http://ftp.jaist.ac.jp/pub/Linux/Gentoo/snapshots/portage-20160306.tar.bz2
  1. un archive stage3 dan portage.
code:tar xvjpf stage3*.bz2
code:tar xvjf /mnt/gentoo/portage-20160306.tar.bz2 -C /mnt/gentoo/usr

6.persiapan sebelum chroot.

code:cp -L /etc/resolv.conf /mnt/gentoo/etc/resolv.conf

dipakai untuk nanti setelah chroot bisa konek internet

code:mount -t proc none /mnt/gentoo/proc
code:mount -o bind /sys /mnt/gentoo/sys
code:mount -o bind /dev /mnt/gentoo/dev
code:chroot /mnt/gentoo /bin/bash
  1. didalam root system gentoo update env.
code:env-update
code:export PS1="(ch) $PS1"
code:mirrorselect -i -r -o >> /mnt/gentoo/etc/portage/make.conf
code:eselect profile list
code:eselect profile set 11
code:nano -w /etc/portage/make.conf

USE="-gtk -gnome qt4 kde dvd alsa cdr"

code:ls /usr/share/zoneinfo
code:cp /usr/share/zoneinfo/Asia/Jakarta /etc/localtime
code:echo "Asia/Jakarta" > /etc/timezone
  1. unduh dan install genkernel untuk ngompile kernel dan membuat initramfs.
code:emerge genkernel

download kernel dengan perintah berikut ini

code:emerge gentoo-sources

kalo saya mindahin dari hasil unduhan saya dan saya unarchive terus di rename menjadi linux

9a kompile kernel.

code:make menuconfig
code:make j2 && make modules_install
code:cp arch/x86_64/boot/bzImage /boot/kernel-4.4.3gentoo

membuat initramfs ini optional

code:genkernel --install initramfs

9b. jika terjadi error ketika ngompile kernel.

code:emerge sys-devel/bc

dan ulangi compile kernel

untuk running modules kernel

code:find /lib/modules/4.4.3gentoo/ -type f -iname '.o' -or -iname '.ko'
  1. unduh dan install grub.
code:emerge grub
code:grub2-install /dev/sda
code:grep -v rootfs /proc/mounts > /etc/mtab
code:grub2-mkconfig -o /boot/grub/grub.cfg
  1. konfigurasi system.
code:nano -w /etc/conf.d/modules

modules_2_6="3c59x"

code:nano -w /etc/fstab

/dev/sdb3 /home ext4 nodev,nosuid 0 2

/dev/sdb2 / ext4 noatime 0 1

#/dev/SWAP none swap sw 0 0

/dev/cdrom /mnt/cdrom auto noauto,ro 0 0

/dev/fd0 /mnt/floppy auto noauto 0 0

code:nano -w /etc/conf.d/hostname

hostname="gentux"

code:passwd
code:nano -w /etc/locale.gen

uncommet yang english

id_ID.UTF-8 UTF-8

code:locale-gen
code:nano -w /etc/env.d/02locale

LANG="id_ID.UTF-8"

LC_COLLATE="C"

code:useradd -m -G users,wheel,audio -s /bin/bash arba
code:passwd arba
code:exit
code:umount /mnt/gentoo/{proc,sys,dev,home}
code:umount /mnt/gentoo
code:reboot
