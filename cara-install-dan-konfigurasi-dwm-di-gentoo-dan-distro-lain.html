CARA INSTALL DAN KONFIGURASI DWM DI GENTOO DAN DISTRO LAIN

Published: Rab 06 April 2016
By arba

In GNU/Linux.

Apa itu dwm(dynamic window manager)? dynamic window manager adalah tilling window manager , yang dinamis dan mudah di kostumisasi dwm dibuild dari bahasa c jadi kita harus mengkompilasinya.

dwm bisa di install di distro gnu/linux, unix. begini cara installnya kalo di gentoo cukup emerge --ask dwm, terus ubah /etc/portage/savedconfig/x11-wm/dwm-6.0.

atau bisa juga dengan mengunduh source codenya untuk lebih optimal install dulu dependenciesnya, 1. libxcb-xinerama0.

  1. libxcb-xinerama0-dev.
  2. libxinerama-dev.
  3. libxinerama1.
  4. xsetroot.
  5. nitrogen.
  6. acpi.
  7. gmrun.

sekarang unduh source dwm kemudian ketikan git clone git clone http://git.suckless.org/dwm di terminal tunggu sampai selesai di unduh kemudian ketik cd dwm untuk berpindah directory ke dwm, kemudian ketikan ln -s config.def.h config.h.

ubah config.h dengan editor text atau pun ide terserah anda kemudian ketikan make && make install atau make -j3 && make install.

ini isi /etc/portage/savedconfig/x11-wm/dwm-6.0 atau config.h

 #!/bin/sh
 # set keyboard layout to Swiss German
 setxkbmap us
 # Load .Xresources
 #file xrdb -merge ~/.Xresources
 Start xbindkeys xbindkeys &
 # Set some defaults
 #export BROWSER=chromium &
 export EDITOR=vim &
 xdg-mime default evince.desktop application/pdf &
 # set backlight
 exec xbacklight -75 &
 # Loop
 while true
 do
    # Set root title
    sh .xsetroot.sh
    # Check battery level
    BATT=$( acpi -b | sed 's/.*[charging|unknown], \([0-9]*\)%.*/\1/gi' )
    STATUS=$( acpi -b | sed 's/.*: \([a-zA-Z]*\),.*/\1/gi' )
    if ([ $BATT -le 5 ] && [ $STATUS == 'Discharging' ]); then
                 # Beep
                 echo -e "\007" >/dev/tty10 && sleep 0.2
                 echo -e "\007" >/dev/tty10 && sleep 0.2
                 echo -e "\007" >/dev/tty10 && sleep 0.2
                 # Blink
                 echo 'on' > /proc/acpi/ibm/light && sleep 1
                 echo 'off' > /proc/acpi/ibm/light
    fi
    # Update every 10s
sleep 1
done &
# Set up rotated dual screen without touching my xorg.conf
xrandr --output DVI-1 --auto --rotate left --pos 0x0 --output DVI-0 --auto --pos 1080x720 --rotate normal
# Read .xsessionrc
#sh ~/.xsessionrc &
# Set wallpaper
feh --bg-scale ~/wallpaper.jpg &
# Set WM name (for Java apps)
wmname LG3D
# run terminal
# xterm -geometry 80x50+494-0 &
# Run dwm
exec dwm
