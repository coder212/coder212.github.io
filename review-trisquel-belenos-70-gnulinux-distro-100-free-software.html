Review Trisquel Belenos 7.0 GNU/Linux Distro 100% Free Software

Published: Sen 26 September 2016
By arba

In Review.

Pernahkan anda mendengar Trisquel GNU/Linux? Kalo belum anda memang benar memilih berkunjung ke situs ini. Trisquel GNU/Linux merupakan Distro GNU/Linux yang mengusung slogan 100% Free Software atau 100% perangkat lunak merdeka artinya Triquel hanya menyertakan paket perangkat lunak yang merupakan perangkat lunak bebas. Semua perangkat lunak, pustaka , driver dan codec yang propriatery dihapus oleh pengembang Trisquel GNU/Linux, oleh karena itu Trisquel GNU/Linux di endorse oleh FSF. Trisquel Menggunakan kernel Linux-Libre yaitu kernel Linux yang sudah dihilangkan kernel modul (driver) yang bukan perangkat lunak bebas.

Oke cukup sampai disini basa-basinya, saya menggunakan Trisquel GNU/Linux kurang lebih 2 bulan. Saya tertarik menggunakan Trisquel GNU/Linux karena rasa penasaran bagaimana rasanya menggunakan kernel linux-lbre apakah berjalan dengan baik pada laptop saya dan sebagainya. Saya menggunakan laptop compaq presario cq-45 dan alhamdulillah berjalan dengan normal, saya menginstall menggunakan installer netinstall sehingga saya membutuhkan koneksi internet untuk menginstallnya. Saya mendapati trisquel tidak terdapat network manager sehingga harus menggunakan perintah terminal, trisquel belenos 7.0 menggunakan gnome 3.8.4 jadul memang tapi masih enak dipandang.

Saya dikejutkan oleh kemampuan Trisquel GNU/Linux dalam memutar file audio berformat mp3 dimana distro yang tidak mengusung semboyan 100% perangkat lunak merdeka saja enggan menyertakan codec mp3. Trisquel menggunakan ABrowser sebagai sarana berselancar di dunia maya benar asyik ABrowser ini. yang saya suka dari ABrowser adalah opsi menon-aktifkan javascript terdapat pada laman utama ABrowser.

Menggunakan Trisquel Belenos 7.0 GNU/Linux untuk penggunaan sehari-hari masih sangat mumpuni dan layak untuk dicoba. yang membuat trisquel istimewa adalah trisqual membawa perangkat lunak yang benar-benar bebas seperti yang diharapkan oleh Richard Matthew Stallman dan FSF.

Demikian ulasan yang saya buat semoga menarik minat pembaca untuk menggunakan perangkat lunak bebas untuk komputasi harian.

performa = 9/10

tampilan= 8/10

kelengkapan perangkat lunak = 8/10
