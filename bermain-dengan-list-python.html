Bermain dengan List Python

Published: Sel 27 September 2016
By arba

In Tutorial.

Sekarang kita akan mencoba bereksperimen atau membuat percobaan melakukan operasi pertambahan pada list yang berisi integer, jadi setiap item dari list a akan di lakukan pertambahan dengan indeks yang sama dari list b. Kemudian hasil ditampilkan dalam bentuk list. Hal yang pertama kita lakukan adalah sebuah fungsi untuk mengambil dua buah list, setelah mendapatkan dua buah list maka dilakukan operasi peertambahan dari tiap item dari 2 list tersebut tepat pada indeks yang sama. Jadi list a indeks 0 akan ditambahkan dengan list b indeks 0.

#!/usr/bin/python3.5
def buatListBaru(ls,als):

    temp = []
    for i in range(0,len(ls):
        temp.append(ls[i] + als[i])

    return temp
if __name__ == '__main__':
    ls = [1,3,5,7]
    als = [2,4,6,8]
    c = buatListBaru(ls,als)
    print (ls)
    print (als)
    print (c)

for digunakan untuk melakukan operasi pertambahan dimulai dari indeks 0 sampai indeks jumlah item dalam list tersebut. kemudian dimasukkan kedalam list temporary dengan menggunakan append jadi append akan menulis kedalam temp dan membentuk list dengan jumlah element sama seperti jumlah perulangan yang dilakukan. kemudian mengembalikan nilai temp kepada variable c.
