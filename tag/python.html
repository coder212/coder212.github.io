Macam-macam Gaya Memasukan Modul pada python

Published: Kam 29 September 2016
By arba

In Tutorial.

tags: Python Import python python in GNU/Linux

ada banyak gaya yang dilakukan untuk memasukan modul kedalam kode sumber dengan bahasa pemrograman python, setidaknya saya menemukan dua gaya yang dipakai untuk memasukan modul. setiap gaya bebas digunakan sesuai dengan keperluan dan gayamu, apakah kamu suka keribetan? Atau mau yang simple? Semua itu bisa dipilih sendiri.

memasukkan library dilakukan dengan menggunakan keywords import disertai nama modul yang akan dimasukan kedalam phyton. Jadi import itu sangat krusial untuk dipelajari tanpa kamu menguasai kemampuan importing maka kamu harus menulis semua modul atau fungsi itu sendiri cukup merepotkan bukan? apalagi buat mereka yang dikejar deadline.

pada python kita hanya bisa memasukan modul buatan kita atau pihak ketiga lainnya hanya pada current directory atau pada directori yang sama tak bisa beda direktori tapi dengan sedikit trik kita bisa memasukan modul dari direktori yang berbeda dengan cara sys.path.insert(0,'/path/direktori/') atau path.insert(0,'/path/direktori/') 0 menunjukan root, gampangkan?
