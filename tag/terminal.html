Perintah terminal yang sering digunakan

berikut adalah perintah - perintah yang sering digunakan didalam terminal gnu/Linux distro, untuk melakukan suatu pekerjaan tertentu.

  1. cp

cp adalah peralatan yang dipakai untuk copy(salin) dan paste(tempel) pada distro gnu/linux cp bisa digunakan untuk salin dan tempel file atau pun folder.

Contoh:

  1. untuk file
code:cp /home/user_name/belajar.txt /home/user_name/arba/

atau

code:cp ~/belajar.txt ~/arba/
  1. untuk folder
code:cp -rf /home/user_nama/arba /home/user_nama/ahmad

*) rf artinya rekursif force artinya mengulangi dan memaksa

atau

code:cp ~/arba ~/ahmad

multiple

code:cp -rf ~/ahmed ~/arba ~/ahmed1 ~/arba1 ~/arba2 ~/ahmed2
  1. mv

mv adalah peralatan yang dipakai untuk memindahkan atau bisa juga me rename suatu file atau folder .

contoh:

  1. untuk file
code:mv /home/user_nama/text.txt /home/user_nama/text.txt1
code:mv /home/user_nama/folder /home/user_nama/test/folder

untuk memindahkan banyak data kedalam sebuah folder

code:mv ~/arba ~/tuyul ~/ahmed masuk

*) dalam situasi folder test udah dibuat

  1. mkdir

mkdir adalah peralatan digunakan untuk membuat folder, dalam membuat folder di gnu/linux distro di dalam terminal tentunya menggunakan perintah mkdir.

Contoh:

  1. membuat folder test.
code:mkdir /home/user_nama/test
code:mkdir ~/test
  1. membuat folder lebih dari satu.
code:mkdir {~/test,~/test1,~/test2}
  1. membuat folder dan sekaligus subfolder secara multiple.
coder:mkdir -p {~/test/subtest,~/test1/subtest1,~/test2/subtest1}
  1. rm

rm adalah peralatan digunakan untuk menghapus file atau pun folder.

contoh:

  1. menghapus file.
code:rm ~/test.txt
  1. menghapus folder.
code:rm -rf ~/test
  1. menghapus folder multiple.
code:rm -rf {~/test,~/test1,~/test2}
  1. menghapus file multiple.
code:rm {~/text1.txt,~/text2.txt,~/text3.txt}
  1. ls.

ls adalah peralatan yang digunakan untukmembuat daftar isi dari suatu folder ataupun directory.

Contoh:

  1. melihat daftar isi folder / direktori.
code:ls -al /home
  1. melihat daftar isi folder / direktori multiple.
code:ls -al {/home,/usr,/boot,/bin}
  1. dd.

dd adalah peralatan yang digunakan untuk membuat bootable ataupun mengcopy filesystem dll

contoh:

code:dd bs=4m if=/home/user_nama/iso/isofile.iso of=/dev/sdb status=progress && sync

cd adalah peralatan untuk yang digunakan untuk berpindah folder.

Contoh:

code:cd ~/Downloads

untuk berpindah dari current direktori kedalam folder Downloads
