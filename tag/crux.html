Apa itu Crux gnu/linux.

crux gnu/linux adalah distribusi gnu/linux yang kecil dan diperuntukan bagi user berpengalaman karena, semuanya di distribusikan dalam bentuk source code. crux gnu/linux enak dipake buat belajar membuat konfigurasi system dan karena sebagian dicompile di komputer kita jadi lebih bagus hasilnya meskipun tak begitu signifikan sih hehe. crux gnu/linux sangat minimalis dan enak untuk dikelola dengan manajer paket prt-get dan pkg-utils lumayan membantu.

jika kamu tidak menemukan paket yang kamu butuhkan kamu dapat mengunduh source code dan mengompilenya didalam systemmu, jadi filosofinya adalah lakukan apa yang kamu suka.

aku mencobanya di laptop compaq presario cq45 sangat memuaskan hasilnya bootingnya cepet, dengan bsd init stylenya yang luar biasa, jadi saya memahami bagaimana system saya boot dan mati proses apa saja yang dibutuhkan untuk on dan off distro saya.

kelebihan:

  1. compile kernel sendiri.
  2. atur konfigurasi system sendiri.
  3. simple dan cepat.
  4. hemat kuota karena cuma 333mb doang.
  5. dapat dipakai dengan berbagai tujuan.

kekurangan:

  1. kompile kernel.

kompile kernel menjadi kelebihan dan sekaligus kekurangannya karena sebagian orang ogah untuk compile kernel, karena ribet.

  1. paket yang disediakan repo crux sangat terbatas.
  2. no office ketika pertama kali install, jadi masih harus install office suite.

Cara Install Crux

buka ini https://crux.nu/Main/Handbook3-2

cara install crux buat partisi dengan cfdisk /dev/sda

Lanjut dengan format tiap partisi

# mkswap /dev/sda5

# mkfs.ext4 /dev/sda6

# mkfs.ext4 /dev/sda2

kemudian mount filesystem

mount partisi baru ke lokasi temporer (/mnt) dan install base packages

# swapon /dev/sda5

# mount /dev/sda6 /mnt

# mkdir /mnt/home

# mount /dev/sda2 /mnt/home

# setup

terus aja ok ok bisa kan

# mount --bind /dev /mnt/dev

# mount --bind /tmp /mnt/tmp

# mount -t proc proc /mnt/proc

# mount -t sysfs none /mnt/sys

# mount -t devpts devpts /mnt/dev/pts

# chroot /mnt /bin/bash

atau

# setup-chroot

kemudian buat fstab

# nano /etc/fstab

/dev/sda6 / ext4 defaults,noatime,nodiratime,discard,errors=remount-ro 0 1

/dev/sda5 swap swap sw,noatime,nodiratime 0 0

/dev/sda2 /home ext4 defaults,noatime,nodiratime,discard,errors=remount-ro 0 1

ctrl + x lalu ctrl + y

kemudia buat rc.conf

FONT=default

KEYMAP=us

TIMEZONE=Asia/Jakarta

HOSTNAME=cucurucrux

SYSLOG=sysklogd

SERVICES=(crond)

generate bahasa system (locales)

# localedef -i en_US -f UTF-8 en_US.UTF-8

# localedef -i id_ID -f UTF-8 id_ID.UTF-8

di lanjutkan dengan

# ln -s /usr/share/zoneinfo/Asia/Jakarta /etc/localtime

kemudian kompile kernelnya

caranya disini

# cd /usr/src/linux-3.12.24

# patch -p1 < ../linux-ptrace-bug.patch

# make menuconfig general setup --->

lalu di save

# make

# make modules_install

# make install

lalu install bootloader (syslinux)

# mkdir /boot/syslinux

# nano /boot/syslinux/syslinux.cfg

DEFAULT crux

PROMPT 0

TIMEOUT 50

LABEL crux

MENU LABEL CRUX

LINUX ../vmlinuz

APPEND root=/dev/sda6 ro

# extlinux -i /booot/syslinux

# dd if=/usr/share/syslinux/mbr.bin of=/dev/sda bs=440 count=1

# passwd

masukin password root baru

# useradd -m -s /bin/bash -G audio,lp,video,wheel,scanner -U nama_user

# passwd nama_user

masukin password user

lalu

# nano /etc/sudoers

nama_user ALL=(ALL) ALL

# exit

# umount /mnt/{dev{/pts,},tmp,proc,sys,home,}

# reboot

kemudian login sebagai root

ketikan nano /etc/prt-get.conf uncomment mv /etc/ports/contrib.rsync.inactive /etc/ports/contrib.rsync

kemudian ketikan perintah ports -u, lalu ports -d untuk melihat perbedaannya, untuk install bisa ketikan prt-get install nama-paket.

kemudian login sebagai user name dengan perintah su user-name.

ketik nano .xinitrc dan tulis exec openbox atau bisa juga openbox-session, sekian review dari saya semoga bermanfaat.
