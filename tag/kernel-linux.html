Cara Compile Kernel Di GNU/Linux.

apa itu kernel? kernel adalah inti dari sistem operasi tiap sistem operasi mempunyai kernelnya masing masing, tergantung bagaimana platform itu diciptakan. platform windows tentu berbeda dengan platform gnu karena sifat gnu sendiri yang bebas.

ok cukup sampai disini saja ngga usah dipanjang lebarin lagi. langkah-langkah compile kernel adalah sebagai berikut:

  1. make mrproper (dilakukan jika udah pernah melakukan compile di source yang saat ini dpakai, kalau belum langsung langkah kedua).
  2. make allyesconfig, allmodulesconfig atau pun menuconfig.
  3. make && make install.

3a. make && make modules_install && make install jika kita pilih config sebagai modules.

catatan :

  1. pastikan sebagai root (#) atau superuser bukan ($) user kalo sebagai user gunakan sudo.
  2. mkinitrd sesuai dengan distro yang digunakan.
  3. update grub jika menggunakan grub berdasarkan dengan distro.

saya menggunakan crux gnu linux jadi saya mengetikan perintah berikut ini saya dalam posisi sebagai root. nano /boot/grub/grub.cfg lalu tekan enter. kemudian saya ketikan " menuentry " CRUX kernel 4.5"{ linux(hd0,msdos1)/boot/vmlinuz-4.5.0-rc6-Asboot root=/dev/sda1 quiet } tergantung type partisi jenis memory yang kamu install gnu/linuxnya terus masuk /dev/sda1 atau yang lainnya root partisinya, bisa juga vmlinuz saja liat apa yang ada dalam /boot. demikian catatan saya semoga bermanfaat.
