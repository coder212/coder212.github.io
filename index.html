Mengenal Lebih Dekat OpenBSD

Published: Kam 06 Oktober 2016
By arba

In Review.

tags: OpenBSD sistem operasi yang berdasarkan BSD

OpenBSD adalah sistem operasi yang dibuat berdasarkan 4.4BSD oleh Theo de Raadt. Sebelumnya Theo mengerjakan dan bertagungjawab dalam proyek porting netBSD kedalam mesin SPARC akan tetapi karena terjadi perbedaan pendapat dengan tim inti pengembangan netBSD, sehingga membuat Theo memutuskan hengkang dari proyek tersebut, dan membuat proyek baru yang dinamakan OpenBSD.

OpenBSD sejak awal dibuat dengan tujuan untuk menciptakan sistem operasi yang aman, untuk tujuan tersebut pada tahun 1996, tim inti pengembangan OpenBSD memeriksa baris per baris kode sumber paket yang akan disertakan dalam rilis OpenBSD guna menciptakan sistem operasi yang aman dari bawaan. Perangkat lunak yang bagus yang dihasilkan proyek OpenBSD yaitu OpenSSH dan lain-lain.

Saya menginstall OpenBSD didalam Qemu emulator lebih mudah daripada yang dikira dan dibayangkan orang bahkan aku rasakan lebih mudah daripada install Archlinux. OpenBSD secara default dan menyarankan penggunanya apalagi yang masih awam untuk memakai paket yang sudah dikompilasi oleh pengembang OpenBSD.

OpenBSD menyediakan menejemen paket pkg yang dapat digunakan untuk menginstall, menguninstall, menghapus dependensi yang orphaned dan memperbarui paket perangkat lunak yang sudah terinstall. Berikut ini adalah contoh-contoh penggunaan pkg:

  1. Install.
code:pkg_add -v wget
  1. Uninstall.
code:pkg_delete -v wget
  1. Update.
code:pkg_add -u wget
  1. Menghapus orphaned dependensi.
code:pkg_delete -a

Lalu bagaimana jika kita ingin menginstall paket dari kode sumbernya jangan khawatir selain kita bisa install dari source code yang kita unduh, OpenBSD juga menyediakan ports yang berisi tree makefile atau digentoo dikenal dengan ebuilds jadi kita tinggal masuk kedalam direktori paket yang akan install kemudian ketikan make install clean atau make install clean PORRTSDIR=/usr/src/ports tentu saja /usr/src/ports merupakan tempat kamu meletakkan atau menaruh ports.

Untuk dapat menggunakan ports kamu harus mengunduh ports.tar.gz dengan cara seperti dibawah ini.

code:cd /tmp
code:ftp ftp://ftp.jaist.ac.jp/pub/OpenBSD/$(uname -r)/ports.tar.gz
code:cd /usr/src
code:tar xzfv /tmp/ports.tar.gz
code:rm -rf /tmp/ports.tar.gz
code:export PORTSDIR='/usr/src/ports/'

untuk mencari paket yang akan dikompilasi dan diinstall

code:cd ports && make search name=lynx

maka akan diberi tahu letak lynx yaitu pada path www/lynx

code:cd www/lynx && make install clean

yang saya rasakan selama menggunakan OpenBSD adalah nyaman-nyaman saja meskipun ada berapa kekurangan yang di miliki OpenBSD yaitu tidak berfungsinya tombol panah dalam applikasi vim atau pun yang lainnya. Dan compiler yang disertakan masih menggunakan gcc versi 4.2 dengan beberapa penambahan untuk meningkatkan performa ketika melakukan kompilasi. Demikianlah Sedikit ulasan OpenBSD dari saya semoga bermanfaat. sumber: .. _openbsd: https://www.openbsd.org/index.html .. _corebsd: http://ikiw.corebsd.or.id/artikel/openbsd
