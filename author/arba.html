Kernel Module Mahluk Apa Itu?

Published: Kam 19 Mei 2016
By arba

In GNU/Linux.

tags: gnu/linux Kernel modules

Apa itu kernel module? Kernel module merupakan potongan dari kode yang dapat diload atau tidak diload kedalam kernel, pada saat itu juga tanpa mematikan atau menghidupkan ulang sistem anda. Mereka ( Kernel Module ) dapat langsung digunakan tanpa melakukan boot ulang sistem. Sebagai contoh, salah satu tipe kernel module adalah perangkat lunak supir ( driver ), yang mengizinkan kernel untuk dapat menjangkau perangkat keras agar terhubung kedalam sistem. Tanpa kernel modul kita akan membangun kernel monolitik dan menambahkan fungsi baru yang dimasukkan kedalam kernel image.

Disamping memiliki kekurangan image kernel yang membengkak juga kamu harus membangun dan memboot sistem secara terus menerus ketika kita membutuhkan fungsi baru. Ketika kamu mencetak sebuah dokumen dengan mesin pencetak ( printer ) yang berperan untuk menjembatani antara mesin pencetak dengan kernel adalah kernel module, alurnya adalah sebagai berikut applikasi --> kernel --> kernel module --> printer.

Untuk melihat kernel modul apa saja yang sudah siap untuk kamu gunakan lsmod, yang mendapatkan informasi dengan membaca berkas /proc/modules. Ketika kernel membutuhkan fitur yang tidak hanya berada dalam wilayah kekuasaan kernel maka kamu harus menggunakan perintah modprobe dan jika anda ingin mengistall kernel module panggil insmod.

Kernel modul adalah solusi agar hardware anda dapat dikenali sekaligus dapat digunakan oleh kernel dan pastinya oleh applikasi atau program yang anda jalankan. Tanpa kernel modul rasanya sangat sulit rasanya meciptakan kernel image yang kecil ukurannya. Kernel modul merupakan perangkat lunak yang berfungsi untuk mengendalikan perangkat keras yang anda miliki. Kernel modul dihadirkan karena untuk mempermudah dalam pembuatan sistem operasi yang kita mampu menambahkan fungsi baru hardware tanpa kita melakukan boot ulang sistem. Kernel modul memanglah bukan hanya untuk printer saja ada juga untuk modem, perangkat audio, perangkat grafis, wifi, usb dan perangkat lainya.
