Bermain-main dengan Python

Published: Sen 26 September 2016
By arba

In Tutorial.

Python adalah bahasa pemrograman tingkat tinggi yang bagus dan menarik untuk dioprek. bahasa ini membutuhkan python intepreter untuk menerjemahkan kode sumber python kedalam bahasa yang di mengerti oleh komputer. Intepreter itu seperti halnya seorang penerjemah yang menerjemahkan apapun yang kita katakan secara langsung. Bahasa Python bisa digunakan di semua sistem operasi yang ada, python bisa dipakai untuk tujuan apa saja.

baiklah kita mulai bermain dengan python 3.5 dan python 2.7, langkah pertama sebelum membuat sebuah kode sumber adalah mencari-cari masalah setelah ketemu pecahkan masalah dengan menggunakan algoritma setelah jadi algoritma baru diterjemahkan kedalam sebuah kode sumber perangkat lunak. permasalahan yang saya hadapi adalah ketika saya memasukkan kata 'gah','tidak','no''0' tidak secara langsung berubah menjadi boolean padahal saya ingin menjadikan kata-kata tersebut diatas menjadi boolean yaitu false. Bolean itu kalo tidak True pasti False atau kalo tidak1 pasti 0. ok yuk langsung dibuat kode sumbernya pada python 3.5, algoritmanya kalian buat sendiri bisa kan.

#!/usr/bin/python3.5

def kataJadiBoolean(kata):
    return not kata.lower() in ('tidak','salah','no','gah','0','false','ora')
if __name__ == '__main__':
    kata = input('masukkan kata-kata yang akan dirubah menjadi boolean ')
    b = kataJadiBoolean(kata)
    iB = b*1
    print(kata,' diubah menjadi boolean ',b,' dan menjadi integer boolean ',iB)

fungsi dari kataJadiBolean(kata) yang kita definisikan sebagai function untuk merubah kata menjadi boolean. Return artinya mengembalikan kata menjadi True atau False namun karene sebelumnya telah di negasikan maka yang dikembalikan adalah lawannya jadi jika bernilai True maka yang kita terima sebagai kembalian adalah False dan jika bernilai False maka yang kita terima adalah True. begitulah penjelasannya mudahkan? ok sekarang kita ubah kode tersebut kedalam python2.7 karena ada diantara pembaca yang masih menggunakan python2.7.

if __name__ == '__main__':
    kata = raw_input('Masukkan kata yang akan di ubah menjadi Boolean ')
    b = kataJadiBoolean(kata)
